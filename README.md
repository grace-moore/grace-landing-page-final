# Grace Moore — Personal Landing Page

A personal landing page built to showcase Grace Moore's professional background, skills, and experience as a Business Analytics and Information Systems student at the University of Iowa. The site serves as a digital portfolio and central hub, providing recruiters, hiring managers, and professional connections with a clean overview of her work history, education, and activities. It was created as part of BAIS 3300: Digital Product Management to demonstrate the full product development cycle — from requirements gathering through design, development, and deployment. The result is a fast, accessible, mobile-responsive static website with no dependencies or backend required.

---

## Features

Planned features and improvements for future versions of this project:

- **Contact Form** — Add a working contact form so visitors can send messages directly from the site without needing to open an email client.
- **Dark Mode** — Implement a light/dark mode toggle that respects the user's system preference and persists across sessions.
- **Project Portfolio Section** — Expand the site with a dedicated section showcasing academic and professional projects, with descriptions, tags, and optional links.
- **Smooth Scroll Animations** — Add subtle entrance animations on scroll to improve visual polish while maintaining accessibility and performance.
- **Custom Domain** — Connect a custom domain name to the Azure Static Web Apps deployment for a more professional public URL.
